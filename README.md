# Microphone-Analog-Front-End-with-Active-Hum-Noise-Rejection

## Overview

This project focuses on the design and simulation of a low-noise analog front-end for amplifying weak differential audio signals while reducing common sources of interference. The complete signal conditioning chain was designed and verified in **LTspice** using analog building blocks such as an instrumentation amplifier, a 50 Hz notch filter, a passive low-pass and an active low-pass filter.

The main objective of this project was to understand how different analog circuits work together to improve signal quality before the signal is converted into digital form using an ADC.

---

## Motivation

Low-level audio signals are easily corrupted by environmental noise. Two of the most common noise sources are:

- **50 Hz mains hum** from AC power lines
- **High-frequency switching noise** generated by power supplies and nearby digital circuits

Instead of using a single amplifier, this project implements a complete analog front-end where each stage performs a specific function to improve the overall signal quality.

---

## Signal Chain

```
Differential Audio Input
          │
          ▼
Passive RC Low-Pass Filter
(Removes high-frequency interference)
          │
          ▼
Instrumentation Amplifier
(Gain ≈ 100)
          │
          ▼
50 Hz Active Notch Filter
(Removes mains hum)
          │
          ▼

(Cutoff ≈ 15 kHz, Gain ≈ 2)
          │
          ▼

```

Overall Voltage Gain:

**≈ 200 V/V (≈46 dB)**

---

## Circuit Blocks

### 1. Passive RC Low-Pass Filter

A passive RC filter is placed at the input to reduce unwanted high-frequency interference before amplification. This helps prevent high-frequency noise from being amplified by the instrumentation amplifier.

---

### 2. Instrumentation Amplifier

A three-op-amp instrumentation amplifier is used to amplify the weak differential input signal.

Features:

- Gain ≈ 100
- High input impedance
- Differential signal amplification
- Good common-mode noise rejection

---

### 3. Active 50 Hz Notch Filter

A notch filter is used to suppress 50 Hz mains interference without significantly affecting the desired audio signal.

This stage removes power-line hum while allowing nearby frequencies to pass with minimal attenuation.

---

### 4. Active Low-Pass Filter

The final stage is an active low-pass filter with:

- Gain ≈ 2
- Cutoff Frequency ≈ 15 kHz

This filter removes high-frequency switching noise and limits the bandwidth before the output is connected to an ADC or any digital processing stage.

# Simulation Test

To evaluate the complete analog front-end, a noisy input signal was created by combining a small differential audio signal with two common interference sources.

### Desired Signal

- Differential Input = **1 mV**
- Frequency = **1 kHz**

### Noise Sources

- 50 Hz mains hum = **2 mV**
- High-frequency switching noise = **10 mV at 50 kHz**

The complete input signal therefore consisted of:

- Desired audio signal
- 50 Hz interference
- 50 kHz switching noise

---

## Simulation Results

Different simulation analyses were performed in LTspice:

- AC Analysis
- Transient Analysis
- Frequency Response
- Gain Verification

The waveform below shows how the signal changes after each stage of the analog front-end.

### Observation

- The passive RC filter reduces high-frequency interference at the input.
- The instrumentation amplifier successfully amplifies the weak differential signal.
- The 50 Hz notch filter significantly suppresses mains hum.
- The active low-pass filter removes most of the remaining high-frequency switching noise.
- The final output retains the desired **1 kHz signal** with very little visible noise.

The final waveform demonstrates that the complete analog front-end effectively amplifies the desired signal while suppressing both low-frequency and high-frequency interference.
